Point Fixe & Zéro

Un point fixe de g est un x tel que g(x) = x. Un zéro de f est un x tel que f(x) = 0.

📖 Théorie
⚙️ Point Fixe
✂️ Dichotomie
📐 Newton
Définitions
Point Fixe : Soit g une fonction continue. On appelle point fixe de g tout réel x tel que :
g(x) = x
Point Zéro : On appelle zéro (ou racine) de f tout réel x tel que :
f(x) = 0
Lien : Trouver un zéro de f revient à trouver un point fixe de g, avec g(x) = x - f(x) ou toute autre forme équivalente.
Théorème du Point Fixe (Banach)
Si g est contractante sur [a,b] (∃k < 1 tel que |g'(x)| ≤ k) et g([a,b]) ⊂ [a,b], alors g admet un unique point fixe dans [a,b] et la suite xₙ₊₁ = g(xₙ) converge vers ce point fixe.
Condition : |g'(x)| < 1 sur l'intervalle
Algorithme Général — Point Fixe
Variables : x0, x1 : Réel ; eps : Réel ; n : Entier Début Lire(x0, eps) n ← 0 Répéter x1 ← g(x0) n ← n + 1 x0 ← x1 Jusqu'à |x1 - x0| < eps Écrire("Point fixe ≈ ", x1) Écrire("Itérations : ", n) Fin
Calcul — Méthode du Point Fixe
Entrez une fonction g(x) telle que g(x) = x au point fixe. Exemples :
g(x) = cos(x) g(x) = (x + 2/x) / 2 g(x) = x - (x*x - 2) / (2*x)
Le résultat s'affichera ici...
nxₙg(xₙ)|xₙ₊₁ - xₙ|
Méthode de Dichotomie
Principe : si f(a)·f(b) < 0, la fonction change de signe sur [a,b]. On divise l'intervalle par 2 à chaque étape.
m = (a + b) / 2 Si f(a)·f(m) < 0 → b = m, sinon a = m Arrêt quand (b - a) < ε
Le résultat s'affichera ici...
nabmf(m)
Méthode de Newton-Raphson
Convergence quadratique. Nécessite f et sa dérivée f'.
xₙ₊₁ = xₙ - f(xₙ) / f'(xₙ)
Le résultat s'affichera ici...
nxₙf(xₙ)xₙ₊₁|xₙ₊₁ - xₙ|

Méthode des Rectangles
On divise [a,b] en n sous-intervalles de largeur h = (b-a)/n.
Gauche : S = h × Σ f(a + i×h) pour i=0..n-1 Droite : S = h × Σ f(a + (i+1)×h) pour i=0..n-1 Milieu : S = h × Σ f(a + (i+½)×h) pour i=0..n-1

Algorithmes d'Optimisation

Trouver les extrema (maximum/minimum) d'une fonction par méthodes itératives.

✂️ Ternaire
∇ Gradient
📖 Théorie
Méthode de Recherche Ternaire
Pour une fonction unimodale sur [a,b], on divise en 3 parties :
m1 = a + (b-a)/3 m2 = b - (b-a)/3 Si f(m1) < f(m2) → b = m2 (minimum) Sinon → a = m1 Arrêt quand (b-a) < ε
Le résultat s'affichera ici...
Descente de Gradient
Algorithme itératif pour trouver un minimum local en suivant la pente négative :
xₙ₊₁ = xₙ - α × f'(xₙ) α = pas d'apprentissage (learning rate)
Converge si α est suffisamment petit.
Le résultat s'affichera ici...
nxₙf(xₙ)f'(xₙ)|Δx|
Théorie — Optimisation
Extremum : x* est un minimum local si f(x*) ≤ f(x) pour x voisin de x*.

Condition nécessaire (ordre 1) :
f'(x*) = 0 (point stationnaire)
Condition suffisante (ordre 2) :
f''(x*) > 0 → minimum local f''(x*) < 0 → maximum local f''(x*) = 0 → inflexion (indéterminé)
Comparaison des méthodes :
— Recherche ternaire : simple, unimodale uniquement, ne nécessite pas f'
— Descente de gradient : nécessite f', sensible au pas α
— Newton : quadratique, nécessite f' et f'', très rapide
Algorithme Ternaire — Pseudocode
Variables : a, b, m1, m2, eps : Réel Début Lire(a, b, eps) TantQue (b - a) > eps Faire m1 ← a + (b - a) / 3 m2 ← b - (b - a) / 3 Si f(m1) < f(m2) Alors b ← m2 Sinon a ← m1 FinSi FinTantQue Écrire("Minimum en x ≈ ", (a+b)/2) Fin
